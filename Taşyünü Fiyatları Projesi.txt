KURAL #2: COĞRAFİ İSKONTO SİSTEMİ
──────────────────────────────────
Merkez: Tuzla/İstanbul (Depo) + Balıkesir (Fabrika)
Mantık: Mesafe + Hacim = Fiyat

Üç Bölge Stratejisi:

🟢 YEŞİL BÖLGE (Nakliye Dahil - En İyi Fiyat)
   İller: İstanbul, Kocaeli, Sakarya, Düzce, Bolu, Balıkesir, Bursa
   Şart: Full kamyon/tır
   Kaynak: Direkt Balıkesir Fabrika
   
🟡 SARI BÖLGE (Depo Parsiyel)
   İlçeler: Tuzla, Pendik, Kartal, Maltepe, Sultanbeyli, 
            Sancaktepe, Gebze
   Şart: Stokta olmalı
   Kaynak: Tuzla Depo
   Fiyat: Fabrika + İç Nakliye + Kar Marjı
   
🔴 KIRMIZI BÖLGE (Nakliye Hariç)
   Durum: Uzak il + Az miktar (örn: Sivas'tan 100 m²)
   Çözüm: "Nakliye Alıcıya Aittir" (EX-WORKS fiyat)
   Alternatif: "Tuzla depomuzdan aldırabilirsiniz" teklifi

🥉 EN UCUZ KOMBİNASYON
──────────────────────
Hedef Kitle: Bütçesi kısıtlı, "ucuza halledeyim" diyen
Strateji: Levha markaysa da, toz grubu yan sanayi

Örnek Kombinasyon:
- Levha: Dalmaçyalı 5cm (Marka)
- Yapıştırıcı & Sıva: All Alçı / TEKNO / ONAT (Yerel)
- Dübel: Standart kalite
- File: 145gr
- Garanti: 2 yıl malzeme

Fiyat Örneği: 196 TL/m² (500 m² için 98.450 TL)

🥈 DENGELİ SİSTEM - "EN ÇOK TERCİH EDİLEN"
──────────────────────────────────────────
Hedef Kitle: Marka isteyenler ama full orijinal istemeyenler
Strateji: Levha + toz grubu farklı markalar (akıllı kombine)
Badge: "EN ÇOK TERCİH EDİLEN" (kartı %5 büyük göster)

Örnek Kombinasyon:
- Levha: Dalmaçyalı 5cm
- Yapıştırıcı & Sıva: Fawori / Optimix (Marka ama farklı)
- Dübel: Fawori standart
- File: 160gr
- Garanti: 5 yıl sistem

Fiyat Örneği: 220 TL/m² (500 m² için 110.000 TL)

KRİTİK SQL SORGULAR
───────────────────

1. Hesaplama Motoru (Müşteri 500 m² girdiğinde):
────────────────────────────────────────────────
-- Dengeli Sistem Paket Fiyat Hesabı
WITH calculation AS (
  SELECT 
    pi.product_id,
    p.name,
    p.brand,
    pi.quantity_per_m2 * 500 AS total_quantity, 
    -- 500 m² için
    
    p.base_price,
    (pi.quantity_per_m2 * 500 * p.base_price) AS item_total
  FROM package_items pi
  JOIN products p ON pi.product_id = p.id
  WHERE pi.package_id = 2 -- Dengeli Sistem ID'si
)
SELECT 
  SUM(item_total) AS package_total_price
FROM calculation;

2. Stok Kontrolü (Hibrit Paket İçin):
─────────────────────────────────────
-- En Ucuz Kombinasyon sunulabilir mi?
SELECT 
  CASE 
    WHEN COUNT(*) = (
      SELECT COUNT(*) 
      FROM package_items 
      WHERE package_id = 1 -- En Ucuz
    ) THEN 'Stokta Mevcut'
    ELSE 'Stokta Eksik - Sadece Fabrika Teslim'
  END AS package_status
FROM warehouse_inventory wi
JOIN package_items pi ON wi.product_id = pi.product_id
WHERE pi.package_id = 1
  AND wi.quantity_in_stock > 0;

3. Lojistik Bar Hesaplaması:
────────────────────────────
-- Müşterinin sepeti tırı ne kadar doldurdu?
SELECT 
  (SUM(cart_items.quantity) / products.packages_in_truck * 100) 
    AS truck_fullness_percent
FROM cart_items
JOIN products ON cart_items.product_id = products.id
WHERE cart_items.user_id = 'current_user'
  AND products.category = 'plate';

4. Vade Farkı Hesaplama:
────────────────────────
-- 90 gün vadeli fiyat
SELECT 
  base_price * POWER(1 + monthly_rate, 3) AS price_with_terms
FROM products p
CROSS JOIN payment_terms pt
WHERE p.id = 123
  AND pt.term_days = 90
  AND pt.is_active = true;

5. Rakip Fiyat Karşılaştırma:
─────────────────────────────
-- Bugünkü fiyatlar vs rakipler
SELECT 
  p.name,
  p.base_price AS our_price,
  cp.competitor_name,
  cp.price AS competitor_price,
  ROUND((p.base_price - cp.price) / cp.price * 100, 2) 
    AS price_diff_percent
FROM products p
JOIN competitor_prices cp 
  ON p.category = cp.product_category
WHERE cp.scraped_at >= CURRENT_DATE
ORDER BY price_diff_percent DESC;
